# WSU Web Design System

We aim to create a more beautiful, accessible, and usable web pressence for the WSU System. Storybook will define all of the unique components, page templates, and design tokens available for the WSU System to adopt and use in their own web sites and applications.

## Tour
### Components

Components are the discrete, usable components to build and create website in the WSU Web Design System.

### Page Templates

Page templates are various different pages, built using the WSU Design System components. 

### Design Tokens

Design tokens defines a consistent pallete of typography, colors, spacing, and other various elements of style.

## Feedback
We are always looking for constructive feedback with regards to the WSU WDS. Please click the feedback icon <Icon name="feedback" /> in the top toolbar, to provide feedback on any specific element of the Design System.
